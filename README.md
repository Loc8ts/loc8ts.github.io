# Loc8ts

The website for **Loc8ts**, a self-hostable inventory management
application.

- Website: https://loc8ts.com
- Container image: `ghcr.io/loc8ts/loc8ts`

## About this repository

This repository holds the generated static output of the Loc8ts website, 
served via GitHub Pages at https://loc8ts.com - source and development live elsewhere. 

It is published automatically and contains the built static output of loc8ts.com.

No issues and contributions should go here - do not open PRs against it.

## Privacy

The site uses no analytics, no cookies, and no third-party trackers.
All fonts and assets are self-hosted and load only from loc8ts.com.
